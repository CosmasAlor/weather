# Weather App

## Overview
This is a simple weather application developed by Cosmas Alor that utilizes the API from weatherapi.com. Users can view weather information for various locations by entering the city name. The application is built using HTML, CSS, JavaScript, and Bootstrap.

## Features
- Weather Information: Users can view current weather conditions, temperature, humidity, and other relevant data for a specified location.
- Search Functionality: Users can search for weather information by entering the name of a city.
- Responsive Design: The application is designed to be responsive and accessible across different devices.

## Technologies Used
- **JavaScript**: Used to fetch weather data from the weatherapi.com API and handle user interactions.
- **HTML**: Markup language for structuring the web pages.
- **CSS**: Stylesheets for enhancing the visual presentation of the application.
- **Bootstrap**: CSS framework for designing responsive and visually appealing user interfaces.

## Installation
1. Clone the repository:
git clone https://github.com/CosmasAlor/weather.git

2. Navigate to the project directory:
cd weather

3. Open `index.html` in your preferred web browser.

## Demo 
Check out the live demo of the Weather App [here](https://cosmasalor.github.io/weather/).

## Usage
1. Open the Weather App in your web browser.
2. Enter the name of a city in the search bar.
3. Press the "Search" button or hit Enter to view the weather information for the specified location.
